# Notebook App

## About

A simple notebook app to manage notes.

## How to run:
## First clone:
```console
git clone git@github.com:AngelDevl/Notebook.git
```
```console
cd Notebook
```
### Production:
```console
docker compose up -d --build
```

### Development:
#### Backend:
```console
cd Backend
```
```console
npm install
```
```console
npm run dev:run
```
#### Frontend:
```console
cd Frontend
```
```console
npm install
```
```console
npm run dev
```

### Testing:
```console
cd Backend
```
```console
npm install
```
```console
npm run test
```
